Backend: pybluez¶

PyMetaWear can use pybluez as BLE communication backend. The package doing the BLE communication is actually gattlib, which can be found on Bitbucket. It is installed as a subpackage to pybluez, a requirement to make documentation buildable on Travis CI.

It can be installed separately as such:

$ pip install pybluez[ble]
